Моделирование 2D

Порядок работы
1. Вкладка "Основные сведения"

Все физические величины задаются в системе СИ

	Кнопка "Выбрать файл геометрии"
	Файл геометрии выглядит аналогично трехмерному случаю
	Пример:
	область 8х16 элементов
	
8 16 //количество элементов по оси х, количество элементов по оси у
0 50 100 150 200 250 300 350 400 //координаты узлов на оси Х (метры)
0 50 100 150 200 250 300 350 400 450 500 550 600 650 700 750 800 //координаты узлов на оси У (метры)

	Кнопка "Выбрать файл свойств"
	аналогично трехмерному случаю 
	свойства задаются в выбранных прямоугольных областях
	первые четыре числа означают границы прямоугольника, 
	следующие шесть
		модуль Юнга
		коэффициент Пуассона
		плотность
		коэффициент при жесткости (<<1)
		коэффициент при массе (<<1)
		коэффициент вязкости(для моделирования жидкости =1 - твердое тело, =0 - жидкость)
	соответственно
	
	Пример:
	область 8х16 элементов
	
0 0 8 16 3400000000 0.27 2100 0.001 0.05 1
0-начальная граница по Х, 
0-начальная граница по У,
8-конечная граница по Х,
16-конечная граница по У,

в этот прямоугольник входят элементы с нулевого по седьмой (0-7)по оси Х и с нулевого по пятнадцатый по оси У
НОМЕРА ЭЛЕМЕНТОВ НАЧИНАЮТСЯ С НУЛЯ

0 2 8 5 65000000 0.27 1800 0.001 0.05 1
в этот прямоугольник входят элементы с нулевого по седьмой по оси Х и со второго по четвертый (2-4) по оси У
НОМЕРА ЭЛЕМЕНТОВ НАЧИНАЮТСЯ С НУЛЯ

	Кнопка "Выбрать Файл нагрузки"
	первая стока:
	длительность приложения силы (сек), период приложения силы (сек), количество точек удара (шт.)
	вторая строка: 
	координаты все точек приложения силы в номерах узлов Х1, У1, Х2, У2, ... через пробел
	третья стока и далее:
	Величина приложенной силы Fx1, Fy1, Fx2, Fy2, ... через пробел 
	
Пример:

0.055 0.005 1 // - длительность приложения силы 0.055сек, период - каждые 0.005сек в течении этого времени, одна точка удара
4 0 	//координаты силы 4й узел по Х, 0 по У(на свободной поверхности) номера узлов считаются с нуля
0 0   	//величина силы в первый раз нуль по обоим направлениям
0 12000	//ведичина силы по Х нуль, по У 12000(Н)
0 0 
0 0  


Сообщения об ошибках - возникающие в процессе чтения файлов ошибки отображаются на консоли 


2. Вкладка "Динамика"

Задаются время моделиования и шаг моделирования в секундах
Объем для записи 
	Задайте номера узлов, для которых должны быть записаны скорости перемещений
	НОМЕРА УЗЛОВ НАЧИНАЮТСЯ С НУЛЯ 

	пример:
	X 0 8 
	У 0 2
	будет записана полоса из двух верхних элементов (узлы 0,1,2 по вертикали, узлы 0-8 по горизонтали)
Нажать на кнопку "динамика" и выбрать файл для записи точек


3. Вкладка "Рисование"

Нажать кнопку "Рисование" и выбрать файл, из которого рисуется модель. 

Если рисуется заранее сохраненный файл, сначала надо задать геометрию во вкладке "Основные сведения",
а только потом рисовать. 

4. Вкладка "Точки наблюдения"

задайте номера точек в узлах элементов, для которых должны быть скорости перемещений. 
УЗЛЫ ЭЛЕМЕНТОВ НАЧИНАЮТСЯ С НУЛЯ

нажать кнопку "Сохранить значения", 
нажать кнопку "Выберите каталог для хранения файлов-точек"
нажать кнопку "записать..."
выбрать файл, из которого будут выбраны файлы-точки. 
если выскакивает предупреждение, что какая-то точка находится вне области, надо поменять эту точку, 
нажать еще раз "сохранить значения", а потом нажать еще раз "Записать".
